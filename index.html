🚀 ScriptInject

Inject custom JavaScript into any website with one line of code. Scripts are stored in your browser's localStorage and persist across visits.

<script src="https://yourusername.github.io/scriptinject/embed.js"></script>

Add this line to any website's <head> tag to enable custom script injection!

📖 How It Works

1. Create Scripts: Write JavaScript code in the editor above and save it.

2. Add to Website: Copy the integration code and paste it into any website's <head> tag.

3. Manage Scripts: Enable/disable scripts, edit them, or delete them. Changes persist in your browser's localStorage.

4. Test Locally: Use the "Test Script" button to see how your script works before deploying.

• Dark mode toggles

• Auto-fill forms

• Custom styling and themes

• Analytics and tracking
